PyCached 0.9.0.1
========================================
Author: Kelezyb
Date: 2014/1/14

Python Key/Value Cache Server

背后的存储引擎为LevelDB, 载入速度和存储数据非常快. 而且产生的数据文件大小也很小
定时存储数据, 保证数据的一致性和安全性
支持String, List, Hash, Set类型的数据存储
